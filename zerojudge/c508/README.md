# c508:去蟲

## 敘述

給你一個長度為N的序列，請你由小到大排序後輸出在第一行
接著除去掉重複的元素後由大到小輸出至第二行


## 說明

輸入:

第一行 N 代表總共有 N 個數字
第二行即為 N 個數字的序列

---

輸出:



## 範例
範例1

```
輸入:
5
2 2 1 3 4
---

輸出:
1 2 2 3 4
4 3 2 1
```
